## 认识项目
> 项目目录  
>  1. public -> 项目打包时所需要要的模板  第三方依赖库  
>  2. src -> 源代码目录，内容在src里面书写
>  3. node_modules -> 项目所依赖的各种插件所放置的位置  
>  4. dist -> 项目编译后的代码目录


## 认识项目中的npm命令
>  
~~~
npm run serve   // 包括的命令可以在package.json中查看到
~~~

## 了解package.json  
>  项目依赖包配置文件  
>  script配置项指定可以运行的命令
>  dependencies配置项指定的是我们的项目运行时需要的依赖插件
>  devDependencies配置项指定的是我们项目在开发时候所需要安装的依赖插件
